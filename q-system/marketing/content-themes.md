# Content Themes

> Define 4-8 themes that rotate on a cycle. Each theme maps to canonical files for source material.

## How to Use

1. Pick 4-8 themes that represent your core positioning angles
2. Assign each to weeks in a rotation
3. `/q-market-plan` uses this rotation to suggest topics
4. `/q-market-create` reads the theme's canonical sources before generating

## Theme Template

```
### Theme: [Name]
- **Core angle:** (the perspective this theme takes)
- **Canonical sources:** (which files in canonical/ to read)
- **Target audience:** (who this resonates with most)
- **Example topics:**
  - [topic 1]
  - [topic 2]
  - [topic 3]
```
