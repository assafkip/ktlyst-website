The Threat
March 19-23, 2026. Attackers retained access from a prior Aqua Security incident. Pushed malicious commits spoofing known contributors. Compromised Trivy v0.69.4 binaries distributed via GitHub Releases, Docker Hub, GHCR, and ECR. 75/76 trivy-action tags force-pushed with credential-stealing payloads. Malware harvested environment variables, SSH keys, cloud credentials, and tokens. Encrypted with AES-256-CBC + RSA-4096 and exfiltrated to Cloudflare Tunnel or fallback GitHub repos. Persistence backdoor installed on developer machines. Attack expanded to npm ecosystem March 22.
